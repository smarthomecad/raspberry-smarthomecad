SmartHome Rasbbery PI 
=====================

Opensource smart home system with ingrations:
- telegram
- HMC5883L compass
- bell 
- onvif cameras
- opencv face detection system
- 12v door lock
- fcm push notifications
- audio speakers

```
npm i
npm start
```

don't forget to add your custom keys (fcm, telegram_api and chat_id)
